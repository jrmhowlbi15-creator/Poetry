# 🎵 Prosody Lab — The Beat of Poetry

An interactive audio-visual tool to **see and hear** stressed syllables in English poetry meters.

## What it does

- **Visual**: Syllables light up one by one — stressed syllables appear larger and brightly coloured
- **Audio**: Stressed syllables play a higher-pitched tone; unstressed ones play softer and lower
- **Two meters supported**:
  - **Iambic** (da-DUM) — e.g. Shakespeare's sonnets
  - **Trochaic** (DUM-da) — e.g. Blake's *Tiger Tiger*

## How to use

1. Select a meter (Iambic or Trochaic)
2. Choose an example line
3. Hit **▶ Play Beat** to hear and see the rhythm

## Tech Stack

- React (with Hooks)
- Web Audio API (for tone generation)
- Inline CSS styles

## Running locally

```bash
npm install
npm run dev
```
